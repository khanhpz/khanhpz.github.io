    Get More Subscribers Using Growth Tools Chatbot

    Growth Tools are powerful features that help bring more users to your bot in different ways on Facebook. There are 4 tools available in BotStar named Messenger Ref URL, Messenger Code, Landing Page and Comment Trigger.

    Messenger Ref URL

    Messenger Ref URL to open the conversation directly with the bot at a designated block. Follow these steps to setup this tool:

    1. Go to Connect>Growth Tool and select Messenger Ref URL.


    2. Under General Settings section, fill in the name for this new tool and turn on the "Active" option to enable it. Under Action section, you can create variables which will be assigned directly to the user, so you can use them in the Flow Editor to identify user.


    3. Click Next to open Trigger site. Here you'll get the URL which allows users to start a conversation with bot. The message sent to users will be set up in Message section as in the below picture. This message can be a new text or an existing block.


    4. Click Next to open Preview site. The Message Preview section displays how the message will show to users. Click Create to finish setting up this tool.


    5. Below is the result:


    Messenger Code

    Messenger Code can be scanned to link the user to the bot immediately.

    1. Go to Connect>Growth Tool and select Messenger Code. Follow these steps to have one:


    2. Under General Settings section, fill in the name for this new tool and turn on the "Active" option to enable it. Under Action section, you can create variables which will be assigned directly to the user, so you can use them in the Flow Editor to identify user.


    3. Click Next to open Trigger site. Here you setup the size and content for Messenger Code.


    4. Click Next to open Preview site. The Message Preview section displays how the message will be shown to users. The Settings Preview section shows the Messenger Code that you've created. Click Download to save the code and click Create to finish setting up this tool.


    Whenever users scan this Code with Messenger App, the designated messeage will immediately popup.

    Landing page

    This tool allows you to displays a checkbox on the form to allow users to choose to receive messages from your bot. Follow these steps to setup it:

    1. Go to Connect>Growth Tool and select Landing Page.


    2. Under General Settings section, fill in the name for this new tool and turn on the "Active" option to enable it. Under Action section, you can create variables which will be assigned directly to the user, so you can use them in the Flow Editor to identify user.


    3. In Trigger site, you can customize the plugin. To be more specific, you'll decide how it's displayed on the landing page and which text to show.


    4. Click Next to open Preview site. The Message Preview section displays how the message will be shown to users. Under Settings Preview section, copy and insert the Plugin Code into the HTML code of your website right before the < /body > tag. Then, copy and insert the HTML Tag into the HTML code of your website anywhere you want to display the plugin. Finally, click Create to finish setting up this tool.


      Note: Before using the code on your website, please add the domain name to the Whitelisted Domains.

    Comment Trigger

    Bot automatically respond to comments and send messages to customers when you create this tool for your bot.

    1. Go to Connect>Growth Tool and select Comment Trigger.


    2. Firstly, please grant Facebook the permission to control your fanpgae message. After that, under General Settings section, fill in the name for this new tool and turn on the "Active" option to enable it. Under Action section, you can create variables which will be assigned directly to the user, so you can use them in the Flow Editor to identify user.


    3. In Trigger site, you can select specific posts to insert plugin and create conditions to filter the audience. If you wish to respond to all replies of comments, turn on Including replies option. And don't forget to write greeting message which will be shown under each comment/reply.


    4. Under Message section, enable Use Opt-in Block option. This allows bot to send Opt-in block after user response greeting message. After that, add filter condition to select which audience this message will be sent to. Lastly, setup the Opt-in Block by composing a text or select an existing block.
